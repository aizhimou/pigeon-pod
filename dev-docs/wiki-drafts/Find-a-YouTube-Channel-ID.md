# Find a YouTube Channel ID

This page shows the fastest way to get a raw YouTube channel ID.

## What This Page Is For

Use this page when:

- channel URL lookup is inaccurate
- you want the most reliable way to add a YouTube channel
- you want to reduce unnecessary YouTube API quota usage

In PigeonPod, using a raw channel ID is usually more accurate than relying on a channel page URL search.

## Prerequisites

Before you start, you need:

- a YouTube channel page
- a browser

## Steps

### 1. Open the target YouTube channel page

Go to the channel you want to subscribe to.

### 2. Open the channel details area

On the channel page:

- click **More** or the channel information area

### 3. Scroll to the bottom

In the expanded details area:

- scroll until you see the share section

### 4. Copy the channel ID

Choose:

- **Share channel**
- then **Copy channel ID**

The copied value is the raw YouTube channel ID you can use directly in PigeonPod.

## Verify

A valid YouTube channel ID usually looks like a stable identifier rather than a normal URL.

After copying it:

1. paste it into PigeonPod when adding a feed
2. confirm the preview resolves to the expected channel

## Why This Is Better Than a Channel URL

Using a raw channel ID is helpful because:

- it is more precise
- it reduces ambiguity
- it usually uses less YouTube API quota than broader URL-based lookup

If a normal channel URL search fails to find the expected channel, try the raw channel ID next.

## Common Failures

### “I cannot find the Copy channel ID action”

Try:

- opening the full desktop channel page instead of a compact mobile view
- expanding the channel details area more carefully
- checking whether YouTube changed the UI placement for the share action

### “I pasted the ID, but the wrong channel still appears”

Check:

- whether you copied a handle or profile URL instead of the actual channel ID
- whether the target page was really the correct channel
