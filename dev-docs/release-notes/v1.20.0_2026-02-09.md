# release 1.20.0

## 💥💥💥

### Changelog
1. Add the capability to update yt-dlp within the application. #90
2. Add auto download delay function, allow user set up a delay for new episodes auto download to maximize the effectiveness of automatic sponsor removal. #64
3. Add global maximum episodes configuration. #89
4. Add opml export function. #91
5. Add a separate chapters.json file and RSS feed chapters according to the podcasting 2.0 license. #84
6. Optimize the display style of yt-dlp custom args to match that of yt-dlp command args. #64
7. Optimize title/description keywords input ui. #87


### 更新日志
1. 添加应用内更新 yt-dlp 的功能。
2. 添加节目自动下载延迟功能。设置节目发布后延迟一定时间再自动下载，最大限度的提高 `--sponsorblock-remove all` 的效果。
3. 添加全局 ”最大剧集保留数“ 配置。
4. 添加将订阅源导出为 OMPL 文件的功能。
5. 按照 Podcasting 2.0 的规范，添加独立的 chapters.json 文件存储和展示播客内容的章节信息。
6. 优化自定义 yt-dlp 参数的显示效果，自动将一个参数项分为一行显示。
7. 优化标题/描述关键词输入 UI
